Purpose of CrowTech

I wanted to take a second and write about the reasoning I decided to create CrowTech. CrowTech was inspired to be a learning environment specifically for Cybersecurity. Throughout my journey to make it all possible I am proving not only to myself but to all the employers out there that I can learn anything. CrowTech's purpose is to show my problem solving, computer language skills(HTML,CSS,Java,Python,SQL), communicational skills through my code! Simulating incident response, ethical hacking, Packet sniffers such as WireShark, Working with different SEIM Tools and following the NIST Guidelines.
